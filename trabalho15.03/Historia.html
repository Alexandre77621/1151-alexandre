Subway Surfers é um jogo móvel de uma "corrida interminável" co-desenvolvido pela Kiloo Games, uma empresa privada com sede na Dinamarca e pela Sybo Games

Data de lançamento: 23 de maio de 2012 ( nove anos atras).

Tem um estilo de jogo eletrônico para um so jogador.

Plataformas: Android, iOS, Microsoft Windows, macOS, Windows Phone, Windows Phone 8, Windows 10 Mobile, Fire OS, Mac OS.

Estilo de jogo: eletrônico de ação rapida, Jogo eletrônico de corrida interminável.

Temple Run é um jogo eletrônico de corrida sem limite, publicado em 2011 pela Imangi Studios. Foi programado e produzido por Keith Shepherd e Natalia Luckyanova, com a participação de Kiril Tchangov.

A data de lançamento deste jogo foi em 3 de agosto de 2011.
